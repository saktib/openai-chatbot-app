# openai-chatbot
A custom conversational AI agent (Chatbot) using openAI APIs.


Setup Server:
> cd  src/server/
> npm install
> node server.js

Setup Client:
> cd  src/client/
> npm install
> npm start


Once the client start, you should be able to upload the PDF file.
Then set the context and enter your question in the chatbox and send.
